Qual pode ser o primeiro passo para resolver esse problema?

Dê uma olhada nos problemas anteriores que você resolveu utilizando while, como o "Contagem Progressiva" e "Contagem Pares"

Mas como posso inserir os dados da sequência?

A melhor opção é criar uma variável para armazenar o valor conforme ele for alterado. Por exemplo: 

collatz = 5
# vamos alterar o collatz para um valor que é o triplo dele mais um
collatz = collatz*3+1
# vamos alterar o collatz para um valor que é metade dele
collatz = collatz/2

Agora basta inseri-lo na lista da mesma forma que fizemos nos exercícios de contagem.

Devo utilizar condicionais, certo?

Sim, aqui vai um exemplo para facilitar, mas não se acostume: 

if temperatura > 30:
    print("Está calor")
elif temperatura < 15:
    print("Está frio")
else:
    print("Está agradável")
